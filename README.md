# my_math_project

## Description
This is a calculator for addition, substraction, multiplication and division. 

## Getting Started 

### Installation
1. Clone the repo 
    git clone https://github.com/danitoneva/my_math_project.git

## Unit tests
The unit_tests.py file defines 4 test functions that test the methods for addition, subtraction, multiplication, and division.

### Runing the Unit Tests
To run all the tests in the tests directory run the following command:
    python .\unit_tests.py
